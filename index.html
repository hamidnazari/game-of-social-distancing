This is a little fun simulation inspired by Conway's Game of Life.

Rules are simple:

  1. The world is a grid of X * Y cells;
  2. An agent (sim person) can appear (blue) in a cell based on "Density Rate";
  3. That agent has a percentage chance of being infected (yellow), "Initial Infected Rate";
  4. Each agent can move in 9 possible directions, picked at random;
  5. If two or more agents end up on the same cell, that's a clos contact event;
  6. If any agents in close contact is infected, the other agents in the same group get infected;
  7. Agents die (black) after a certain number of days being infected;
  8. Dead agents are buried (disappear) after a certain number of days after dying.
